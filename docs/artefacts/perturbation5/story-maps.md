# Story Map mise à jour

## Objectif

Mettre à jour la story map du projet pour intégrer les nouveaux axes liés au succès national de l'application.

## Activités principales

| Activité utilisateur | Fonctionnalités associées | Axe |
|---|---|---|
| Accéder à la plateforme | Connexion, inscription, page d'accueil | core |
| Réviser un cours | Consultation des matières, fiches, quiz | core |
| Interagir avec l'IA | Poser une question, recevoir une explication personnalisée | core |
| Utiliser le site avec un handicap | Navigation clavier, lecteur d'écran, contrastes | a11y |
| Changer de langue | Sélecteur de langue, traduction de l'interface | i18n |
| Recevoir une réponse IA dans sa langue | Paramètre de langue envoyé au LLM | i18n |
| Utiliser la plateforme en période de forte charge | Cache, tests de charge, architecture scalable | scale |

## Priorités MVP

| Priorité | Éléments |
|---|---|
| Must | Accessibilité des pages principales, changement de langue, test de charge |
| Should | Réponses IA multilingues, cache, amélioration des performances |
| Could | Fournisseur IA de secours, optimisation avancée |
| Won't | Fonctionnalités secondaires non nécessaires à la perturbation |
