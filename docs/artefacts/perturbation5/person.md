# Personne élargie

## Lucia García

- Âge : 17 ans
- Pays : Espagne
- Profil : lycéenne préparant ses examens
- Langue principale : espagnol
- Situation : malvoyante, utilise un lecteur d'écran

## Besoins

Lucia doit pouvoir utiliser EduTutor IA en espagnol et accéder aux contenus malgré son handicap visuel.

Elle a besoin :

- d'une interface traduite ;
- de quiz traduits ;
- de réponses IA dans sa langue ;
- d'une navigation au clavier ;
- de contrastes suffisants ;
- de textes alternatifs pour les images ;
- d'une compatibilité avec les lecteurs d'écran.

## Frustrations

Lucia ne peut pas utiliser correctement une application uniquement pensée pour des élèves français voyants.

Elle peut être bloquée si :

- les boutons ne sont pas lisibles par un lecteur d'écran ;
- les contrastes sont trop faibles ;
- le site ne fonctionne pas au clavier ;
- les contenus ne sont disponibles qu'en français.

## Impact sur le produit

Cette personne justifie les nouveaux axes de la perturbation :

- [a11y] accessibilité RGAA ;
- [i18n] internationalisation ;
- [scale] capacité à accueillir un grand nombre d'utilisateurs.
