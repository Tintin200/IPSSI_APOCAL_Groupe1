# Perturbation 5 — Le succès qui change tout

Ce dossier contient les artefacts mis à jour pour répondre à la perturbation 5 du projet Apocalypse / EduTutor IA.

## Contexte

L'application connaît un fort succès national. Des millions d'élèves se connectent et l'État français souhaite adopter la plateforme comme référence pour les lycées.

Cette évolution doit disposer de trois nouveaux axes prioritaires :

- accessibilité RGAA ;
- internationalisation ;
- scalabilité et résilience.

## Objectif du livrable

L'objectif est de montrer que l'équipe sait piloter le passage à l'échelle du projet avec des artefacts agiles mis à jour, une analyse des risques et un suivi du planning.

## Artefacts fournis

- Vision board mis à jour
- Persona élargie
- Story map mise à jour
- Product backlog repriorisé
- Release planning mis à jour
- Next sprint backlog
- Analyse de risques
- Pilotage burndown / burnup
